# Frontend (React + Vite)

This is the React frontend of the RBAC MERN application.

## Development
```bash
npm install
npm run dev
```

Environment
- `VITE_API_URL` must point to the backend (e.g., `http://localhost:5000` in dev or your Render URL in prod).

## Build
```bash
npm run build
```

For full project details and backend/API docs, see the root README.
